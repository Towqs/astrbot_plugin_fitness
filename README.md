# 🏋️ AstrBot 智能健身教练插件 v2.0.11

QQ群智能健身教练，通过自然对话提供个性化健身指导，集成RPG游戏化、周期化训练、成就系统等丰富功能。

## 🆕 最近更新

- **训练任务模板升级** — 周期计划会结合性别、目标、器材生成更清晰的训练分化，减少每天任务雷同。
- **用户自定义计划** — 新增 `安排今天训练`、`安排明天训练`、`安排本周训练`、`清空本周训练`，用户可以直接覆盖自己的训练安排。
- **动作视频引导** — `今日计划` 会按动作附带 B 站动作教学搜索链接，可在 WebUI 配置中关闭。
- **健身帮助入口** — 群聊艾特机器人发送 `健身帮助`，或直接使用 `健身帮助` 命令，可查看插件用法说明；其他泛化帮助词不再触发，减少与其他插件冲突。
- **管理员私聊查询** — 管理员可在私聊中通过群号查询指定群的用户档案和全员档案，群内操作方式保持兼容。

## ✨ 功能特性

### 核心功能
- **自然对话建档** — AI 分步引导收集用户信息，建立个性化健身档案
- **训练计划生成** — 根据用户目标、器材、体质生成个性化训练计划
- **每日打卡** — 记录训练数据，自动计算经验值
- **训练质量评估** — 根据打卡内容记录计划完成度和训练负荷
- **定时提醒** — 基于APScheduler精确提醒，支持AI个性化提醒
- **动态调整** — 根据用户状态（生病/受伤/疲劳）自动调整计划
- **渐进式信息收集** — 日常对话中自然收集训练经验、饮食习惯等
- **AI综合分析** — 收集足够信息后自动生成训练方向建议
- **双模型支持** — 高成本模型用于对话，低成本模型用于提醒
- **5种教练人格** — 热血教练/温柔姐姐/毒舌损友/军训教官/游戏NPC

### v2.0 新增功能
- **🎯 周期化训练** — 4-8周训练周期，自动渐进超负荷，减载周提醒
- **😴 疲劳评估** — 打卡时自动评估疲劳度，高疲劳触发减量建议
- **🥗 饮食打卡** — 自然语言记录饮食，AI估算热量和蛋白质
- **🏆 成就系统** — 12种成就，打卡/升级/连续训练等行为触发解锁
- **📈 进步检测** — 自动检测训练量提升、连续打卡等进步并给予反馈
- **👤 用户画像** — 综合分析训练偏好、活跃时段、疲劳趋势
- **📊 群周报** — 每周日自动发送群健身排行榜和数据统计
- **📅 周六体重反馈** — 每周六定时询问体重变化和训练感受
- **🔄 补卡机制** — 支持补签前一天遗漏的打卡（经验减半）
- **💬 主动回复** — 教练以可配置概率主动回复群聊消息，更有存在感

### RPG游戏化
- **等级系统** — 经验值升级，从Lv.1到无上限
- **称号体系** — 根据等级自动获得称号
- **随机事件** — 打卡时触发暴击训练、宝箱掉落等
- **QQ群头衔同步** — 升级自动更新群专属头衔（需机器人为群主）

## ⚙️ 配置项

在 AstrBot WebUI 插件配置页面可设置：

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 生效群白名单 | 限制插件生效的群号，逗号分隔 | 空（所有群） |
| 教练人格风格 | 5种预设 + 自定义 | 热血教练 |
| 启用定时打卡提醒 | 每日打卡提醒 | 开启 |
| 默认提醒时间 | 新用户默认提醒时间 | 17:30 |
| 启用群头衔同步 | 升级时更新群头衔 | 开启 |
| 启用随机事件 | 打卡触发随机事件 | 开启 |
| 启用加练建议 | AI判断是否建议加练 | 开启 |
| 启用成就系统 | 成就解锁和经验奖励 | 开启 |
| 启用饮食打卡 | 饮食记录和热量估算 | 开启 |
| 启用动作视频引导 | 今日计划附带 B 站动作教学搜索链接 | 开启 |
| 允许用户自定义训练计划 | 允许用户手动覆盖自己的训练安排 | 开启 |
| 周六体重反馈时间 | 每周六询问体重的时间 | 10:00 |
| 群周报发送时间 | 每周日发送周报的时间 | 20:00 |
| 高成本LLM模型 | 对话/建档/计划用模型 | 默认模型 |
| 低成本LLM模型 | 提醒/简单任务用模型 | 纯模板 |
| 启用主动回复 | 教练概率性主动回复群消息 | 关闭 |
| 主动回复概率(%) | 主动回复触发概率，1-100 | 5 |

## 🌐 环境要求

- AstrBot >= 4.20.0
- NapCatQQ（aiocqhttp 协议）
- 机器人需为群主才能设置群头衔

## 📦 安装

通过 AstrBot WebUI 插件市场搜索「智能健身教练」安装，或手动克隆到 `data/plugins/` 目录：

```bash
cd data/plugins/
git clone https://github.com/Towqs/astrbot_plugin_fitness.git
```

## 💡 使用说明

1. 安装插件后在 WebUI 配置页面设置生效群号等参数
2. 在群里发送「健身注册」，机器人会通过私聊引导你完成建档（保护隐私）
3. 机器人在私聊中分步采集身高体重等信息，完成后群里自动通知
4. 建档完成后，每天发送「打卡」记录训练
5. 发送「饮食记录 /食物内容」可直接记录饮食；只发送「饮食记录」后，也可以在 10 分钟内继续发食物内容
6. 日常和教练聊天即可获得训练建议、计划调整等服务
7. 有 bug 可以加我 QQ：2775089081
