# 🏋️ AstrBot 智能健身教练插件

QQ群智能健身教练，通过自然对话提供个性化健身指导，集成RPG游戏化系统。

## 功能特性

- **自然对话建档** — AI 分步引导收集用户信息，建立个性化健身档案
- **训练计划生成** — 根据用户目标、器材、体质生成个性化训练计划
- **每日打卡** — 记录训练数据，自动计算经验值
- **RPG游戏化** — 等级系统、称号、闯关任务、随机事件
- **QQ群头衔同步** — 升级自动更新群专属头衔（需机器人为群主）
- **定时提醒** — 基于APScheduler精确提醒，支持AI个性化提醒
- **动态调整** — 根据用户状态（生病/受伤/疲劳）自动调整计划
- **渐进式信息收集** — 日常对话中自然收集训练经验、饮食习惯等
- **AI综合分析** — 收集足够信息后自动生成训练方向建议
- **双模型支持** — 高成本模型用于对话，低成本模型用于提醒
- **5种教练人格** — 热血教练/温柔姐姐/毒舌损友/军训教官/游戏NPC

## 用户指令

| 指令 | 说明 |
|------|------|
| `健身注册` | 手动触发建档流程 |
| `打卡` | 开始今日打卡 |
| `我的档案` | 查看个人健身档案 |
| `今日计划` | 查看今天的训练计划 |
| `我的计划` | 查看训练方向分析和近期计划 |

## 管理员指令

| 指令 | 说明 |
|------|------|
| `查看档案 @某人` | 查看指定用户的完整档案 |
| `查看所有档案` | 查看本群所有已建档用户概览 |

## 配置项

在 AstrBot WebUI 插件配置页面可设置：

- 生效群白名单
- 教练人格风格（5种预设 + 自定义）
- 打卡提醒开关
- 群头衔同步开关
- 随机事件开关
- 加练建议开关
- 高成本/低成本 LLM 模型配置

## 环境要求

- AstrBot >= 4.20.0
- NapCatQQ（aiocqhttp 协议）
- 机器人需为群主才能设置群头衔

## 安装

通过 AstrBot WebUI 插件市场搜索安装，或手动克隆到 `data/plugins/` 目录。
